Possibly meant _import instead of _local.

Noncompliant Code Example

_method a.m()
  _local log_stream << !terminal!
  _proc@do_something()
    _local log_stream
    log_stream.write("hello from do_something")
  _endproc
_endmethod

Compliant Solution

_method a.m()
  _local log_stream << !terminal!
  _proc@do_something()
    _import log_stream
    log_stream.write("hello from do_something")
  _endproc
_endmethod
